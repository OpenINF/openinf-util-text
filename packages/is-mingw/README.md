# is-mingw

[![npm version](https://img.shields.io/npm/v/is-mingw.svg)](https://www.npmjs.com/package/is-mingw)

> Check if the process is running inside MinGW

## Installation

```sh
npm install --save is-mingw
```

## Usage

```js
const isMinGW = require('is-mingw');

console.log(isMinGW());
```
